Secure Authentication System

A full-stack authentication platform built with React, Flask, and MongoDB, featuring JWT-based login, role-based access control, and OTP verification. The system ensures secure and scalable user management for modern web applications.

What I Did

I built a complete authentication system that connects a React + TypeScript frontend to a Flask (Python) backend, secured by JWT tokens and integrated with MongoDB for persistent data storage. The platform supports registration, login, and secure session handling through encrypted cookies and stateless authentication.

I also implemented role-based access control (RBAC) for admin and user-level permissions, as well as OTP verification for enhanced login security. The system follows clean architectural principles for scalability and modularity, with well-documented endpoints and a flexible design suitable for production deployment.

Key Contributions

  • Developed a secure full-stack authentication system using React, Flask, and MongoDB.
  • Implemented JWT authentication with tokens stored in secure cookies.
  • Added OTP verification for enhanced login protection.
  • Built role-based access control (admin/user) with scalable backend logic.
  • Ensured modular code structure, clear separation between frontend and backend, and easy deployment on cloud platforms.
