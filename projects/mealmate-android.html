MealMate (Android App)

A feature-rich Android application designed to simplify recipe discovery, meal planning, and grocery management. Built with Kotlin, Firebase, and Room Database, the app provides both online and offline access with a clean Material Design interface.

What I Did

I developed MealMate from the ground up as a native Android app that helps users discover recipes, create grocery lists, and manage meal tasks collaboratively. The app integrates the TheMealDB API for live recipe data and uses Firebase for cloud storage and real-time synchronization across devices.

I implemented a responsive and modular architecture using the MVC pattern, separating logic between activities, fragments, and adapters. Users can save favorite meals, view nearby grocery stores via geotagging, and access stored data offline through the Room Database.

Key Contributions

  • Developed a fully functional native Android app using Kotlin and Firebase.
  • Integrated TheMealDB API for live recipe search and categorization.
  • Implemented offline storage with Room Database and cloud backup with Firebase Realtime Database.
  • Added user authentication, profile customization, and favorite meal management.
  • Applied Material Design principles for a consistent and intuitive user interface.
